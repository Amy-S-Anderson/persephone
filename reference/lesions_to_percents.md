# Calculate lesion prevalence by age interval

Calculate lesion prevalence by age interval

## Usage

``` r
lesions_to_percents(cemetery_df, group_vars)
```

## Arguments

- cemetery_df:

  Data frame of cemetery data with age and lesion columns

- group_vars:

  Character vector of grouping variable names

## Value

Data frame with Lesion_Percent by Age_Interval and groups
